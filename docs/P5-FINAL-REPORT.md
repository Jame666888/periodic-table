## 四、功能接入方式

### 趋势图
- **位置**：元素详情页（`element.html?z=XX`）侧边栏 → "📈 属性趋势图"卡片
- **操作**：
  1. 选择属性（电负性/原子半径/电离能等）
  2. 选择范围（同周期 / 同族）
  3. 图表自动更新，当前元素用醒目颜色标记

### 丰度对比图
- **位置**：元素详情页侧边栏 → "🌍 元素丰度对比"卡片
- **操作**：
  1. 勾选对比环境（🌍 地壳 / ☀️ 太阳系 / 🧬 人体）
  2. 图表和表格自动更新
  3. 使用对数坐标展示（log₁₀），便于对比极端差异的丰度

## 五、已知问题与待优化项

### 1. 丰度数据覆盖不全
- **问题**：`abundance-data.js` 中部分微量元素（如 Tc、Pm、超铀元素）的丰度数据缺失（标注为 0 或 null）
- **影响**：这些元素的丰度图会显示"暂无数据"
- **建议**：逐步补充稀有元素和超铀元素的丰度估算值（来自核合成理论）

### 2. 趋势图属性数据依赖 `elements.js`
- **问题**：趋势图从 `ELEMENTS` 数组中读取属性值，若某属性的 `null` 较多，图表会断裂
- **影响**：部分元素的同周期/同族趋势图可能数据不完整
- **建议**：逐步补全 `elements.js` 中缺失的属性数据

### 3. 丰度图使用对数坐标可能增加理解难度
- **问题**：对数坐标对非专业用户可能不直观
- **建议**：添加"切换为线性坐标"按钮（需在 Chart.js 中动态修改 `ticks.callback`）

### 4. 趋势图未保存用户偏好
- **问题**：每次刷新页面，趋势图会重置为默认属性（电负性）和范围（同周期）
- **建议**：使用 `localStorage` 保存用户上次选择的属性和范围
