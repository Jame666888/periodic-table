## 4️⃣ 数据验证模块

### 📁 文件
- **主文件**：`js/data-validation-enhanced.js`
- **类名**：`DataValidator`
- **权威数据**：`AUTHORITATIVE_DATA`（全局变量）

### 📊 权威数据源
- **主要来源**：NIST Chemistry WebBook (https://webbook.nist.gov/chemistry/)
- **数据元素数**：20个（H、He、Li、Be、B、C、N、O、F、Na、Mg、Al、Si、P、S、Cl、K、Ca、Fe、Cu、Zn、Ag、Au）
- **验证字段**：
  1. 相对原子质量（atomicMass）
  2. 熔点（meltingPoint）
  3. 沸点（boilingPoint）
  4. 密度（density）
  5. 电负性（electronegativity）

### 🔬 验证算法
1. **加载本地数据**：从 `ELEMENTS` 数组读取
2. **对比权威数据**：逐个字段比较
3. **容差设置**：
   - 相对原子质量：±0.01
   - 熔点/沸点：±5°C
   - 电负性：±0.1
4. **严重度分级**：
   - `high`：差值超过阈值10倍
   - `medium`：差值超过阈值
   - `low`：差值在可接受范围

### 📝 验证报告
运行 `validator.validateAll()` 生成 Markdown 格式报告，包含：
1. **统计概览**：总元素数、一致数、差异数、无数据数
2. **差异详情**：按严重度分级（严重、中等、轻微）
3. **建议**：修正建议、数据来源、定期验证

### 💾 导出功能
- **JSON格式**：`validator.getResultsJSON()`
- **Markdown格式**：`validator.generateReport()`
