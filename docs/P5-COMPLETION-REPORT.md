## 一、P5 功能清单

### ✅ P5-1：3D 电子轨道可视化
**文件**：`js/p5-orbital.js`（~200 行）

**功能**：
- s 轨道：球形可视化（SphereGeometry）
- p 轨道：哑铃形（x/y/z 三个方向）
- d 轨道：四叶草形（d_z² 和 d_xy）
- 自动旋转动画
- 坐标轴辅助

**技术**：
- 使用 Three.js（CDN 加载，已在 `element.html` head 中引用）
- 轻量级实现，无需额外依赖

**接入方式**：
- `element.html` 侧边栏新增"🌀 电子轨道 3D"卡片
- 点击按钮调用 `createOrbitalPanel(el)`

---

### ✅ P5-2：属性热力图
**文件**：`js/p5-heatmap.js`（~180 行）

**功能**：
- 8 种属性着色：电负性、原子半径、密度、熔点、沸点、原子量、电子亲和能、电离能
- 周期表格子按数值自动着色
- 颜色图例显示
- 重置功能

**注意**：
- `index.html` 中已有内联热力图代码（392-482 行），功能重复
- 建议：保留内联版本（已与 HTML 结构集成），或统一使用 `p5-heatmap.js`
- 当前状态：两个版本共存，按钮调用内联 `toggleHeatmap()`

---

### ✅ P5-3：增强元素对比
**文件**：`js/p5-compare.js`（~150 行）、`compare.html`（~250 行）

**功能**：
- 并排对比 2-4 个元素
- 属性差异高亮（绿色=最高值，红色=最低值）
- 雷达图对比（Canvas 绘制）
- CSV 导出
- 响应式设计

**接入方式**：
- 首页"⚡ 元素对比"按钮链接到 `compare.html`
- 支持 URL 参数 `?z=1,6,8` 直接对比指定元素

---

### ✅ P5-4：交互式学习测验
**文件**：`js/p5-quiz.js`（~200 行）

**功能**：
- 18 道精选题目（基础、进阶、方程式、应用、安全）
- 选择题题型
- 即时反馈 + 解析
- 错题回顾
- 学习进度保存到 localStorage
- 得分评级（优秀/良好/需加强/继续学习）

**接入方式**：
- 首页新增"📝 知识测验"按钮
- 点击调用 `showQuizPanel()`
