  - 过渡金属（Sc, Ti, V, Cr, Mn, Fe, Co, Ni, Cu, Zn, Y, Zr, Nb, Mo, Tc, Ru, Rh, Pd, Ag, Cd, Hf, Ta, W, Re, Os, Ir, Pt, Au, Hg）- 80+个方程式

- **反应类型**（如"与酸反应"、"歧化反应"）
- **反应条件**（如"室温"、"加热"、"450°C, 15-30MPa"）
- **说明**（反应机理、工业应用、注意事项）

## 🔬 任务 #45：添加晶体结构图片

### 完成内容
- **文件**：`js/crystal-structure-svg.js`
- **功能**：SVG晶体结构渲染器
- **支持结构**：
  1. **BCC（体心立方）** - Na, K, Fe, Cr, W等
  2. **FCC（面心立方）** - Al, Cu, Ag, Au, Ni等
  3. **HCP（六方密堆积）** - Mg, Ti, Zn, Cd等
  4. **金刚石型** - C（金刚石）, Si, Ge等
  5. **简单立方** - 少数元素

### 渲染效果
- 生成 200x200 SVG示意图
- 元素类别配色方案（碱金属→红色，过渡金属→蓝色等）
- 标注配位数
- 可嵌入元素详情Markdown中

### 集成方式
在 `element.html` 中引用 `crystal-structure-svg.js`，v4.0补丁自动在"物理性质"部分后嵌入晶体结构SVG。

## ⚖️ 任务 #46：交互式配平练习

### 完成内容
- **文件**：`js/equation-balancing.js`
- **功能**：交互式化学方程式配平练习模块

### 功能特性
1. **显示方程式**：系数位置显示为输入框
2. **实时验证**：点击"检查答案"按钮，验证方程式是否配平
3. **提示功能**：点击"提示"按钮，显示配平提示
4. **显示答案**：点击"显示答案"按钮，显示正确系数

### 测试用例
创建了3个练习题：
1. `Fe + HCl → FeCl₂ + H₂`（系数：1, 2, 1, 1）
2. `Al + O₂ → Al₂O₃`（系数：4, 3, 2）
3. `N₂ + H₂ → NH₃`（系数：1, 3, 2）

### 技术实现
- 使用JavaScript类 `EquationBalancingExercise`
- 化学方程式解析器 `EquationParser`
- 支持常见化学式（如FeCl₂, H₂SO₄等）
