六、Phase 2：权威引用与可验证溯源体系

6.1 引用体系设计

每条数据均应标注来源，形成可追溯的信息链：

  • 数据引用标注格式：[来源: 机构, 年份, 版本]，如 [IUPAC CIAAW, 2024, 原子量标准]
  • DOI/URL 超链接：文献引用直接链接到原文 DOI（如 doi.org/10.1515/pac-2022-0802）
  • CAS 号：每个元素的常见化合物标注 CAS 注册号（如硫酸 CAS 7664-93-9）
  • PubChem CID：化合物标注 PubChem 化合物 ID
  • 更新时间戳：每个数据页面标注上次更新日期，保证时效透明

6.2 关键参考出版物

出版物用途获取方式
CRC Handbook of Chemistry and Physics (第 105 版+)绝大多数理化常数hbcponline.com（付费）
IUPAC "Atomic weights of the elements" (Pure Appl. Chem.)原子量权威值ciaaw.org（免费）
NIST-JANAF Thermochemical Tables热力学数据NIST（免费）
Lange's Handbook of Chemistry综合化学数据McGraw-Hill
中国《危险化学品目录》(2022 版)中国法规合规应急管理部官网
ILO/WHO ICSC 国际化学品安全卡安全与应急ilo.org/dyn/icsc/（免费多语言）

6.3 数据时效性保障机制

  • 原子量：每 2 年一次复查（IUPAC 更新周期）
  • 同位素半衰期：跟随 NuDat 3.0 数据库更新
  • 超重元素：跟踪 IUPAC/JWP 确认公告
  • 安全法规：跟随 GB 标准及 GHS 更新
  • 价格/产量：跟随 USGS Mineral Commodity Summaries（年度报告）

七、Phase 3：工业、安全与合规数据

7.1 GHS 危害分类（每个元素的常见形态）

7.2 毒理与暴露数据

数据项说明
GHS 象形图火焰/腐蚀/骷髅/环境危害等标准图标
信号词危险(Danger) / 警告(Warning)
H 编码 (Hazard Statements)如 H228 易燃固体、H314 造成严重皮肤灼伤
P 编码 (Precautionary Statements)如 P210 远离热源、P280 戴防护手套
UN 编号危险货物联合国编号（如 UN 1017 氯气）
运输分类危险货物类别与包装等级

7.3 工业生产数据

  • 工业制备方法：主要工艺路线（如氯碱法、拜耳法、Kroll 法）及对应化学方程式
  • 全球年产量：源自 USGS Mineral Commodity Summaries
  • 主要生产国/企业：全球产能分布
  • 价格趋势：近年市场价格（如 ¥/kg 或 $/tonne），标注来源（如 SMM 上海有色网）
  • 主要矿物：含该元素的天然矿物列表（来源：Mindat.org / IMA）
  • 地壳丰度：ppm 或 mg/kg（来源：CRC）

7.4 专业级泄漏与应急处置

  • ERG 指南编号：对应北美应急响应指南 (Emergency Response Guidebook)
  • 隔离距离：小/大泄漏的初始隔离与防护距离
  • 个人防护装备(PPE)：具体的呼吸器类型（如自给式正压呼吸器）、防护服材质要求
  • 灭火介质：适用与禁用的灭火剂
  • 中和/吸收方法：具体化学处理方案（含方程式）
  • 废水/废气处理：排放标准（GB 8978 / GB 16297）+ 处理方法

八、Phase 4：科研级深度数据

8.1 同位素完整数据

从目前的radioactive: true/false 二元标注，升级为：

  • 每种同位素的质量数、精确质量（amu）
  • 天然丰度（%），带不确定度
  • 半衰期（含单位：y, d, h, min, s, ms, μs, ns）
  • 衰变模式（α, β⁻, β⁺, EC, SF, IT, p, n, 2β 等）
  • 衰变能（MeV）
  • 子体核素
  • 衰变链可视化（对铀、钍等天然放射系）
  • 来源：NuDat 3.0 (BNL) + IAEA LiveChart of Nuclides

8.2 光谱数据

  • 原子发射光谱：特征谱线波长(nm)及强度（来源：NIST ASD）
  • X 射线光谱：Kα、Kβ、Lα 线的能量(keV)（来源：NIST）
  • XPS 结合能：常见轨道电子结合能(eV)（来源：NIST XPS Database）
  • NMR 性质：核自旋、自然丰度 NMR 核、旋磁比、参考化合物化学位移

8.3 晶体学数据

对每种元素的常见同素异形体/晶型：

  • 空间群 (Space group)
  • 晶格参数 a, b, c (Å) 及 α, β, γ (°)
  • 单胞原子数 Z
  • 配位数
  • x, y, z 原子分数坐标
  • 来源：COD / ICSD / Pearson's Crystal Data

8.4 研究前沿

  • 每个元素的研究前沿方向（如："2024 年 Nature/Science 相关论文"）
  • 关键综述文献引用（DOI 链接，一年内）
  • 中国研究团队突出贡献标注
  • 开放研究问题/未解决问题
  • 超重元素最新合成进展（如 Z=119, 120 的合成尝试）

九、Phase 5：交互功能与可视化升级

9.1 高级搜索与筛选

  • 多条件筛选器：按族、周期、物态、磁性、导电性、价格区间等筛选
  • 属性排序：按原子量/密度/电负性/熔点等属性排序展示
  • 正则 + 模糊搜索：支持化学式（如 "Fe₂O₃"）直接搜索
  • 中文拼音搜索：输入 "qing" → 匹配"氢"

9.2 元素对比工具

  • 并排对比 2–4 个元素的全部属性
  • 差异高亮（Δ 计算）
  • 对比结果可导出为 PDF/表格

9.3 3D 可视化

  • 晶体结构：使用 Three.js 渲染 fcc/bcc/hcp 等晶格 3D 模型
  • 电子轨道：s/p/d/f 轨道形状 3D 展示
  • 分子结构：常见化合物分子 3D 模型（可旋转/缩放）

9.4 数据可视化

  • 属性热力图：周期表上按电负性/原子半径/密度着色
  • 趋势图：族/周期内属性变化趋势线图（Plotly/Chart.js）
  • 丰度图：地壳/太阳系/人体元素丰度对比图
  • 同位素图：核素图（Z vs N，标注稳定性）

9.5 API 与数据导出

  • RESTful JSON API：/api/elements、/api/elements/{Z}
  • 数据导出：CSV、JSON、XML 格式
  • 嵌入卡片：提供 iframe/Web Component 嵌入到其他网站

  • 便于增量更新和版本控制
